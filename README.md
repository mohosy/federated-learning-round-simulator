# Federated Learning Round Simulator

A frontend simulator for federated optimization rounds with client heterogeneity, dropout, and adversarial behavior.

## Highlights
- FedAvg-style update aggregation.
- Configurable clients, local epochs, dropout, and adversarial rate.
- Live global-loss trend and per-round client update distribution.
- Convergence score and participation tracking.

## Stack
- HTML5 Canvas
- Vanilla JavaScript
- Responsive CSS dashboard

## Run
Open `index.html` in your browser.
